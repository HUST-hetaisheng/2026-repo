# 📊 美赛 2026 项目仓库（2026-repo）

本仓库为 **一支来自华中科技大学的团队** 在 **2026
年数学建模竞赛（MCM/ICM）** 中的项目仓库，用于统一管理竞赛相关的
**代码、数据、模型文档、论文与可视化结果**。

仓库内容涵盖从问题分析、模型假设、算法实现，到论文撰写与结果展示的完整流程，适合作为
**参赛记录、成果归档与后续学习参考**。

------------------------------------------------------------------------

## 📁 仓库结构说明

    ├── code/                      # 核心算法与实验代码
    ├── data/                      # 原始数据与预处理数据
    ├── figs/ / figures/           # 绘图结果与可视化图像
    ├── polish/                    # 论文美化与展示材料
    ├── prompt/                    # AI / ChatGPT 提示词记录
    ├── 2026_MCM_Problem_C.pdf     # 官方题目 PDF
    ├── 2026_MCM_Problem_C.txt     # 题目文本版
    ├── Memo.pdf                   # 建模过程备忘与总结
    ├── main.tex                   # LaTeX 论文主文件
    ├── mcmthesis-demo.pdf         # LaTeX 模板示例输出
    ├── plot_rank_pct.py           # 数据分析与绘图脚本
    ├── 模型假设说明.md            # 建模假设与合理性说明
    ├── 第二题思路.md              # 第二问解题思路
    ├── 第三问思路.md              # 第三问解题思路
    ├── task1-论文重整补全记录.md  # 论文修改与完善记录
    └── README.md                  # 项目说明文档

------------------------------------------------------------------------

## 🧠 项目内容概述

-   竞赛题目整理（PDF / 文本）
-   数学模型建立与假设分析
-   算法实现与实验代码
-   结果可视化与统计分析
-   LaTeX 论文模板与完整论文
